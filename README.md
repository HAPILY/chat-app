# 「出会い系アプリ」

## 環境構築
- git clone
取り込みたいディレクトリでgit cloneする。
```
$ git clone https://github.com/THitokuse/chat-app.git
```

- docker-compose up　でdockerを立ち上げる。
```
$ docker-compose up -d
```

- docker /bin/sh にログインする。
```
$ docker exec -it chat-app_vue_app_1 /bin/sh
```

- yarnを立ち上げる。
```
/chat-app # yarn start
```
